# ScanFlow — AI Implementation Contract (MANDATORY)

## Critical Directive

This document is a UI/UX enhancement specification only.

You are NOT authorized to redesign, refactor, restructure, optimize, simplify, replace, remove, merge, rename, or alter any existing application logic, workflows, navigation architecture, processing pipelines, state management, data flow, business rules, storage mechanisms, routing behavior, scanner functionality, image processing functionality, PDF generation functionality, sharing functionality, or feature architecture.

Your responsibility is strictly limited to improving the visual presentation and user experience while preserving 100% functional parity.

---

# Non-Negotiable Rule

If a requested UI improvement requires modifying application logic, feature behavior, workflow sequence, route structure, processing pipeline, or architectural flow:

DO NOT IMPLEMENT IT.

Instead:

1. Preserve the existing implementation.
2. Improve only the visual layer.
3. Keep all functionality operating exactly as before.

The application must behave identically before and after UI improvements.

2. Preserve architecture.
3. Preserve workflow.
