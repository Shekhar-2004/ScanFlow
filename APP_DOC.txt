# ScanFlow - Project Master Document

## Project Overview

### Project Name

ScanFlow

### Tagline

A lightweight, offline-first document scanner focused on speed, simplicity, privacy, and excellent user experience.

### Vision

Most modern scanner applications have become bloated with:

* Cloud integrations
* Mandatory accounts
* AI upselling
* Analytics
* Advertisements
* Excessive permissions

ScanFlow aims to provide the core functionality users actually need:

* Scan documents
* Improve readability
* Generate PDFs
* Share documents

All while remaining lightweight, fast, and privacy-friendly.

The goal is not to compete with enterprise products like Adobe Acrobat.

The goal is to become the best scanner app for everyday users.

---

# Core Principles

## 1. Offline First

The application must work fully without internet access.

No feature in the MVP should require a network connection.

---

## 2. Fast Launch

Target startup time:

< 2 seconds on mid-range devices.

---

## 3. Lightweight

Target APK size:

Version 1:
25-40 MB

Future versions:
Remain under 70 MB whenever possible.

---

## 4. Privacy Respecting

No account creation.

No analytics.

No mandatory cloud sync.

No user tracking.

---

## 5. Beautiful Design

Clean Material 3 interface.

Minimal clutter.

Large touch targets.

Focus on usability.

# MVP Scope

The MVP must only include the following features.

## Included

### Home Screen

Functions:

* Create new scan
* View recent documents

---

### Camera Screen

Functions:

* Open camera
* Capture document image

---

### Editor Screen

Functions:

* Crop image
* Rotate image
* Apply filters

Filters:

* Original
* Grayscale
* High Contrast
* Black & White

---

### PDF Preview Screen

Functions:

* Preview generated PDF
* Rename document
* Save document

---

### Share Screen

Functions:

* Share PDF
* Open Android share sheet

Examples:

* WhatsApp
* Gmail
* Telegram
* Drive
* Bluetooth

---

## Excluded from MVP

Do NOT build these until MVP is complete.

* OCR
* AI enhancement
* Cloud sync
* User accounts
* Collaboration
* Document search
* Backup systems
* Multi-device sync

Version 1 is considered successful if a user can:

1. Open the app.
2. Scan a document.
3. Improve readability.
4. Export a PDF.
5. Share the PDF.

Without:

* Internet
* Account creation
* Advertisements

Everything else is secondary.
